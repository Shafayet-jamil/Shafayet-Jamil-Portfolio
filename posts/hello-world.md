# Hello World — Why I Started This Blog

Every developer has a "hello world" moment. You know, that first time you run a program and see output on the screen. It feels tiny, but it changes something inside you.

This blog is my hello world moment for writing in public.

## Why I'm Doing This

I've been learning to code for a while now, and I've noticed something: the things I understand best are the things I've tried to explain to someone else.

There's a concept called the **Feynman Technique** — if you can't explain something simply, you don't really understand it. I want to test myself.

So I'm going to write about:

- Things I learn in class and on my own
- Projects I build and the problems I run into
- Concepts that confused me and how I finally understood them
- Interesting ideas I come across

## Who Is This For?

Honestly? Mostly for me.

But if you're also a beginner in CS or web development, I hope some of these posts save you the hours I spent confused. I'll try to explain things the way I wish someone had explained them to me — simply, with examples, without assuming too much.

## What to Expect

I'm not an expert. I'm learning. Some posts might have mistakes. If you spot one, I'd genuinely love to know.

I'll write about HTML, CSS, JavaScript, CS fundamentals, and whatever else I'm figuring out. No fixed schedule. Just whenever I have something worth sharing.
